# C++ OOP Projects

This folder contains my major Object Oriented Programming project developed in C++ using a multi-file structure.

---

## **Fitness Trainer System**

A console-based application designed to simulate a personal fitness trainer platform using modular and reusable OOP design.

### **Core Features**
- Member profile management  
- Personalized exercise planning  
- Nutrition planning and tracking  
- Progress tracking system  

### **OOP Concepts Demonstrated**
- Multi-class architecture  
- Header and source file separation  
- Encapsulation and abstraction  
- Modular program design  
- Real-world problem modelling  

### **Why This Project Matters**
This project represents my ability to move beyond small programs and build a complete, structured application using Object Oriented Programming principles.
